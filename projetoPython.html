Análise desportiva

Como a minha ultima posição profissional tinha uma vertente tão grande de analise de dados fiquei curioso e decidi investir tempo de forma a compreender melhor o que se passa no mundo futebolistico.
Apesar de ser um projeto sem fim e eu estar constantemente a aprender, aqui esta o que reuni até agora:

Como esta organizado o projeto

O primeiro passo que temos de fazer é retirar o dados de alguma fonte. Dessa forma foi escolhido o site dos meus resultados.
Existem dois tipos de extrações, uma com todas as ligas juntas num ficheiro só que se chama "dados_futebol_todos" e outro "dados_futebol" no qual está dividido entre folhas do mesmo ficheiro. De forma a conseguirmos ter ambas as vertentes dos dados, caso queiramos fazer uma analise macro ou mais especifica a alguma liga.
