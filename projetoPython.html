Análise desportiva

Como a minha ultima posição profissional tinha uma vertente tão grande de analise de dados fiquei curioso e decidi investir tempo de forma a compreender melhor o que se passa no mundo futebolistico.
Apesar de ser um projeto sem fim e eu estar constantemente a aprender, aqui esta o que reuni até agora:

Como esta organizado o projeto

O primeiro passo que temos de fazer é retirar o dados de alguma fonte. Dessa forma foi escolhido o site dos meus resultados. O processo é totalmente automatico desde a abertura de páginas com o site até a criação do ficheiro excel com os dataframes com todos os dados.
Existem dois tipos de extrações, uma com todas as ligas juntas num ficheiro só que se chama "dados_futebol_todos" e outro "dados_futebol" no qual está dividido entre folhas do mesmo ficheiro. De forma a conseguirmos ter ambas as vertentes dos dados, caso queiramos fazer uma analise macro ou mais especifica a alguma liga.

O que esta em cada folha:

- Folha Jogos_Futuro: Todos os jogos que vão decorrer. Com uma visualização rápida das classificações em que as equipas estão e uma conta rápida da diferença entre classificações.
- Folha Jogos_Passado: É a grande fonte de informação. Todas as contas feitas a frente tem a base nestes resultados. É uma simples recolha sem nenhuma conta nem analise, apenas auxilio.
- Folha Classificacao: Neste momento já existe uma grande quantidade de contas feita em relação com os dados dos resultados da folha "Jogos_passado". São também um alimento para as análises que vem na proxima instância onde os titlos são bastante diretos aos calculos que estão a ser feitos.
- Folha Classificacao_proximo_jogo: Aqui é a minha parte favorita e onde vai ocorrer todo o resultado dos dados sobre os quais estamos a trabalhar sobre. Neste momento sobre a Classificacao da equipa esta exposto o próximo adeversário e se joga em casa ou fora.
      Dado que é um projeto quase sem fim, tenciono que nesta folha sejam adicionados mais dados recorrentes da informação que coletamos anteriormente.
      Avaliação do potencial de uma equipa para marcar golos com base em padrões e contexto do jogo.
